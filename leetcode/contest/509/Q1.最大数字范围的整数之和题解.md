# 最大数字范围的整数之和

## 1. 题目概述

- **题目名称**：Q1. 最大数字范围的整数之和
- **链接**：[3982. 最大数字范围的整数之和](https://leetcode.cn/problems/sum-of-integers-with-maximum-digit-range/)
- **来源**：LeetCode 第 509 场周赛 Q1
- **难度**：简单
- **标签**：数学、模拟

**题意简述**：

给定整数数组 `nums`。一个整数的**数字范围** = 其最大数字 − 最小数字。返回 `nums` 中所有数字范围等于数组中**最大数字范围**的整数之和。

**约束条件**：

- `1 <= nums.length <= 100`
- `10 <= nums[i] <= 10^5`

## 2. 示例

**示例 1**

```text
输入：nums = [5724,111,350]
输出：6074
解释：
  5724：max=7, min=2, 范围=5
  111： max=1, min=1, 范围=0
  350： max=5, min=0, 范围=5
  最大范围=5，5724+350=6074
```

**示例 2**

```text
输入：nums = [90,900]
输出：990
解释：两者范围均为 9（max=9, min=0），90+900=990。
```

## 3. 解题思路

### 3.1 暴力思路

逐个整数提取数位，算 max digit 和 min digit，求差。找全局最大范围，再求和。签到题，直接模拟。

### 3.2 核心观察：两遍扫描

![两遍扫描：求范围 → 求和](../../images/max_digit_range_flow.svg)

1. **第一遍**：对每个 `nums[i]` 计算数字范围 `range_i = max_digit - min_digit`，记录全局最大 `max_range`
2. **第二遍**：对所有 `range_i == max_range` 的 `nums[i]` 求和

### 3.3 数位提取

对整数 `x` 逐位提取（`x % 10` 取末位，`x /= 10` 前进），跟踪最大和最小数字：

```text
max_d = 0, min_d = 9
while x > 0:
    d = x % 10
    max_d = max(max_d, d)
    min_d = min(min_d, d)
    x //= 10
range = max_d - min_d
```

### 3.4 示例演算

`nums = [5724, 111, 350]`：

| i | nums[i] | 数位 | max | min | 范围 |
|---|---------|------|-----|-----|------|
| 0 | 5724 | 5,7,2,4 | 7 | 2 | 5 |
| 1 | 111 | 1,1,1 | 1 | 1 | 0 |
| 2 | 350 | 3,5,0 | 5 | 0 | 5 |

最大范围 = 5，求和 = 5724 + 350 = 6074 ✓

## 4. 复杂度分析

- **时间复杂度**：`O(n · d)`。`n` 个数，每个数最多 `d` 位（`d = log10(max(nums)) <= 6`）。
- **空间复杂度**：`O(1)`。仅用几个变量。

> 💡 可合并为一遍扫描：先算所有范围存数组，同时记录 max_range，再求和。或直接存 `(range, value)` 列表。

## 5. 参考代码

### C++

```cpp
class Solution {
public:
    int maxDigitRange(vector<int>& nums) {
        int maxRange = -1;
        vector<int> ranges;
        for (int x : nums) {
            int maxD = 0, minD = 9;
            int t = x;
            while (t > 0) {
                int d = t % 10;
                maxD = max(maxD, d);
                minD = min(minD, d);
                t /= 10;
            }
            ranges.push_back(maxD - minD);
            maxRange = max(maxRange, maxD - minD);
        }
        int ans = 0;
        for (int i = 0; i < nums.size(); i++) {
            if (ranges[i] == maxRange) ans += nums[i];
        }
        return ans;
    }
};
```

### Python

```python
class Solution:
    def maxDigitRange(self, nums: list[int]) -> int:
        ranges = []
        max_range = -1
        for x in nums:
            digits = [int(c) for c in str(x)]
            r = max(digits) - min(digits)
            ranges.append(r)
            max_range = max(max_range, r)
        return sum(nums[i] for i in range(len(nums)) if ranges[i] == max_range)
```

## 6. 边界情况与易错点

1. **所有数字相同**：如 `111`，范围 = 0，所有数都计入求和。
2. **`nums[i] = 10`**：数位 1,0，范围 = 1−0 = 1。注意 `10` 是最小输入。
3. **单元素数组**：答案 = 该元素本身。
4. **`max_digit` 初始化为 0、`min_digit` 初始化为 9**：因为数位范围是 0-9，这是安全的初值。
