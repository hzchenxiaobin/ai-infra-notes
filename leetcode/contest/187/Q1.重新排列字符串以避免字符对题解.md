## 1. 题目概述

- **题目名称**：Q1. 重新排列字符串以避免字符对
- **链接**：[101116. 重新排列字符串以避免字符对](https://leetcode.cn/problems/rearrange-string-to-avoid-character-pair/)
- **来源**：LeetCode 第 187 场双周赛 Q1
- **难度**：简单
- **标签**：贪心、计数、字符串

**题意简述**：
给定一个字符串 `s` 和两个不同的小写字母 `x`、`y`。要求把 `s` 重排成一个排列 `t`，使得 `t` 中所有 `y` 都出现在所有 `x` 之前。返回任意一个合法排列即可。

**约束条件**：
- `1 <= s.length <= 100`
- `s` 仅由小写英文字母组成
- `x`、`y` 为小写英文字母且 `x != y`

## 2. 示例

**示例 1**

```text
输入：s = "aabc", x = "a", y = "c"
输出："cbaa"
解释："cbaa" 是原串排列，且所有 'c' 都在所有 'a' 之前。
```

**示例 2**

```text
输入：s = "dcab", x = "d", y = "b"
输出："cabd"
解释："cabd" 是原串排列，且所有 'b' 都在所有 'd' 之前。
```

**示例 3**

```text
输入：s = "axe", x = "o", y = "x"
输出："axe"
解释：'o' 根本没出现，条件自动满足。
```

## 3. 解题思路

### 3.1 暴力思路
枚举 `s` 的所有排列并检查条件，复杂度 `O(|s|!)`，显然不可行。

### 3.2 核心观察：三段式构造
只需要保证**所有** `y` **在所有** `x` **之前**，其他字符之间的相对顺序可以任意。因此可以直接把最终字符串分成三段：

1. 第一段：所有的 `y`；
2. 第二段：除 `x` 以外的其他字符；
3. 第三段：所有的 `x`。

这样任何 `y` 都在最前面，任何 `x` 都在最后面，自然满足条件。

![三段式构造](images/bc187_q1_rearrange_segments.svg)

### 3.3 算法流程
1. 遍历 `s`，统计 `y` 的个数、其他非 `x` 字符的序列、`x` 的个数。
2. 按 `y + 其他非 x 字符 + x` 拼接成答案。

### 3.4 示例演算
对 `s = "aabc"`、`x = 'a'`、`y = 'c'`：
- `y` 有 1 个：`'c'`；
- 其他非 `x` 字符有 1 个：`'b'`；
- `x` 有 2 个：`'aa'`。
拼接得到 `"cbaa"`。

## 4. 算法细节

- 由于只关心 `x` 和 `y`，可以用一个计数数组或简单地把字符分成三类。
- 第二段“其他字符”内部顺序任意；为了代码简单，遍历原串时把既不是 `x` 也不是 `y` 的字符直接追加即可，这样也天然保证是原串的排列。

## 5. 正确性证明

**定理**：按 `y` 全放前面、非 `x` 非 `y` 字符放中间、`x` 全放后面的构造一定满足题目要求。

**证明**：
- 该构造使用的字符数与 `s` 中每个字符的个数完全相同，因此它是 `s` 的一个排列。
- 最终串中 `y` 只出现在第一段，`x` 只出现在第三段，因此任意一个 `y` 的位置都小于任意一个 `x` 的位置，即所有 `y` 都在所有 `x` 之前。
- 证毕。 ∎

## 6. 复杂度分析

- **时间复杂度**：`O(n)`，其中 `n = s.length`，只需遍历字符串一次。
- **空间复杂度**：`O(n)` 用于存储答案；若忽略输出，则额外 `O(1)`。

## 7. 参考代码

### C++

```cpp
class Solution {
public:
    string rearrangeString(string s, char x, char y) {
        string ypart, mid, xpart;
        for (char c : s) {
            if (c == y) ypart += c;
            else if (c == x) xpart += c;
            else mid += c;
        }
        return ypart + mid + xpart;
    }
};
```

### Python

```python
class Solution:
    def rearrangeString(self, s: str, x: str, y: str) -> str:
        ypart = [c for c in s if c == y]
        mid = [c for c in s if c != x and c != y]
        xpart = [c for c in s if c == x]
        return "".join(ypart + mid + xpart)
```

## 8. 边界情况与易错点

1. `x` **或** `y` **在** `s` **中未出现**：构造依然有效。若 `y` 未出现，第一段为空；若 `x` 未出现，第三段为空。
2. `s` **长度为 1**：直接返回 `s`。
3. **第二段字符顺序**：不需要排序或保持原序，任意顺序都合法；但必须是原串中的字符，不能漏或重复。
