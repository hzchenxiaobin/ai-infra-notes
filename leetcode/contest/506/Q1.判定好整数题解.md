# 判定好整数

## 1. 题目概述

- **题目名称**：Q1. 判定好整数
- **链接**：[3959. 判定好整数](https://leetcode.cn/problems/check-good-integer/)
- **来源**：LeetCode 第 506 场周赛 Q1
- **难度**：简单
- **标签**：数学、模拟

**题意简述**：

给定正整数 `n`。令 `digitSum` 为 `n` 的各位数字之和，`squareSum` 为 `n` 的各位数字平方之和。若 `squareSum - digitSum >= 50`，则称 `n` 为**好整数**，返回 `true`；否则返回 `false`。

**约束条件**：

- `1 <= n <= 10^9`

## 2. 示例

**示例 1**

```text
输入：n = 1000
输出：false
解释：数字为 1,0,0,0。digitSum=1, squareSum=1, 差=0 < 50。
```

**示例 2**

```text
输入：n = 19
输出：true
解释：数字为 1,9。digitSum=10, squareSum=82, 差=72 >= 50。
```

---

## 3. 解题思路

### 3.1 暴力思路

逐位提取数字，累加 `digitSum` 与 `squareSum`，最后判断差值。签到题，直接模拟即可。

### 3.2 核心观察：化简公式

![好整数公式化简](images/check_good_integer_table.svg)

将差值变形：

```text
squareSum - digitSum = Σ(d²) - Σ(d) = Σ(d² - d) = Σ d·(d-1)
```

每个数字 `d` 对差值的**贡献**为 `d·(d-1)`：

| 数字 d | d·(d-1) 贡献 |
|--------|-------------|
| 0 | 0 |
| 1 | 0 |
| 2 | 2 |
| 3 | 6 |
| 4 | 12 |
| 5 | 20 |
| 6 | 30 |
| 7 | 42 |
| 8 | 56 |
| 9 | 72 |

> 💡 **关键观察**：数字 `0` 和 `1` 贡献为 0（不增加差值）。一个 `9` 贡献 72 已超过 50，一个 `8` 贡献 56 也够。因此好整数只需含一个 `8`/`9`，或若干小数字的组合（如 `6+5=50`，`7+4=54` 等）。

### 3.3 算法流程

1. 逐位提取 `n` 的数字（`n % 10` 取末位，`n /= 10` 前进）
2. 累加 `digitSum += d`，`squareSum += d*d`
3. 循环结束后判断 `squareSum - digitSum >= 50`

![各数字 d 的贡献 d·(d-1) 条形图](images/good_integer_contributions.svg)

## 4. 算法细节

1. **数位提取**：`n % 10` 取末位，`n /= 10` 前进一位，循环至 `n == 0`。
2. **累加**：`digitSum += d`，`squareSum += d*d`，两变量同步维护。
3. **公式化简**：`squareSum - digitSum = Σ(d²-d) = Σ d·(d-1)`，数字 0/1 贡献 0，可提前剪枝（若累计差值已达 50 直接返回 `true`）。
4. **数据类型**：`d*d` 最大 81，10 位最多 810，`int` 足够；C++ 用 `long long` 仅为保险。

---

## 5. 正确性证明

**引理 1**：`squareSum - digitSum = Σ d·(d-1)`。

**证明**：`squareSum = Σ d²`，`digitSum = Σ d`，差 = `Σ(d² - d) = Σ d·(d-1)`。∎

**定理**：算法返回 `n` 是否为好整数。

**证明**：由引理 1，差值等于各位 `d·(d-1)` 之和。算法逐位累加 `digitSum` 与 `squareSum`，循环覆盖 `n` 的所有位（`n >= 1` 保证至少一位）。最终 `squareSum - digitSum >= 50` 的判定与定义一致。∎

---

## 6. 复杂度分析

- **时间复杂度**：`O(log n)`。`n` 最多 10 位，逐位处理。
- **空间复杂度**：`O(1)`。仅用两个累加变量。

## 7. 参考代码

### C++

```cpp
class Solution {
public:
    bool checkGoodInteger(int n) {
        long long digitSum = 0, squareSum = 0;
        while (n > 0) {
            int d = n % 10;
            digitSum += d;
            squareSum += (long long)d * d;
            n /= 10;
        }
        return squareSum - digitSum >= 50;
    }
};
```

### Python

```python
class Solution:
    def checkGoodInteger(self, n: int) -> bool:
        digit_sum = square_sum = 0
        while n > 0:
            d = n % 10
            digit_sum += d
            square_sum += d * d
            n //= 10
        return square_sum - digit_sum >= 50
```

---

## 8. 边界情况与易错点

1. **`n = 1`**：单数字 1，差值 = 1 - 1 = 0 < 50，返回 `false`。
2. **整数溢出**：`d*d` 最大 81，10 位最多 810，`long long` 安全，但 C++ 中 `d*d` 若 `d` 为 `int` 不会溢出（81 远小于 INT_MAX）。
3. **公式化简**：`d²-d = d(d-1)`，数字 0/1 贡献为 0，可提前剪枝（若已 >= 50 直接返回）。
