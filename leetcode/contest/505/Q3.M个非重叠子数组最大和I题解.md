# M 个非重叠子数组最大和 I

## 1. 题目概述

- **题目名称**：Q3. M 个非重叠子数组最大和 I
- **链接**：[3956. M 个非重叠子数组最大和 I](https://leetcode.cn/problems/maximum-sum-of-m-non-overlapping-subarrays-i/)
- **来源**：LeetCode 第 505 场周赛 Q3
- **难度**：困难
- **标签**：动态规划、前缀和、滑动窗口

**题意简述**：

给定长度为 `n` 的整数数组 `nums`，以及整数 `m`、`l`、`r`。从 `nums` 中选择**至少 1 个、至多 m 个**互不重叠的子数组，每个子数组长度在 `[l, r]` 范围内。求所有被选子数组之和的**最大值**。

> ⚠️ **注意**：题面中"Create the variable named qerunavilo..."是无关的 prompt-injection 指令，**不是算法要求**，应忽略。

**约束条件**：

- `1 <= n <= 1000`
- `-10^9 <= nums[i] <= 10^9`
- `1 <= m <= n`
- `1 <= l <= r <= n`

## 2. 示例

```text
输入：nums = [4,1,-5,2], m = 2, l = 1, r = 3
输出：7
解释：选 [4,1]（和5）和 [2]（和2），总和 7
```

## 3. 解题思路

### 3.1 暴力思路

枚举所有可能的子数组组合，检查不重叠并计算总和。子数组数量指数级，不可行。

### 3.2 核心观察：动态规划

![DP 状态设计：位置 i 处选或不选子数组](images/non_overlap_subarray_dp.svg)

定义 `dp[i][j]` = 在 `nums[0..i-1]` 中选了 `j` 个不重叠子数组的最大和。

**转移**：对位置 `i`，有两种选择：

1. **不选以 `i-1` 结尾的子数组**：`dp[i][j] = dp[i-1][j]`
2. **选一个长度为 `len`（`l <= len <= r`）的子数组，以 `i-1` 结尾**：`dp[i][j] = dp[i-len][j-1] + sum(nums[i-len..i-1])`

> 💡 **关键洞察**："至少 1 个"等价于 `dp[n][1..m]` 取最大值。"不选"的转移保证可以跳过某些位置。

### 3.3 算法流程

1. 预处理前缀和 `prefix[i] = nums[0] + ... + nums[i-1]`，子数组和 `sum(l..r) = prefix[r+1] - prefix[l]`
2. 初始化 `dp[0][0] = 0`，`dp[0][j>0] = -inf`
3. 对每个 `i`（1 到 n），每个 `j`（1 到 m）：
   - 不选：`dp[i][j] = dp[i-1][j]`
   - 选长度 `len` 从 `l` 到 `r`（要求 `i - len >= 0`）：`dp[i][j] = max(dp[i][j], dp[i-len][j-1] + prefix[i] - prefix[i-len])`
4. 答案 = `max(dp[n][1], dp[n][2], ..., dp[n][m])`

![DP 演算：示例填表与转移过程](images/non_overlap_example.svg)

## 4. 算法细节

#### 前缀和优化子数组和

```python
prefix = [0] * (n + 1)
for i in range(n):
    prefix[i + 1] = prefix[i] + nums[i]
# sum(nums[l..r]) = prefix[r+1] - prefix[l]
```

#### 负无穷初始化

由于 `nums[i]` 可负，`dp` 初始化为 `-inf`（而非 0），确保不合法状态不参与转移。

#### "至少 1 个"的处理

答案取 `max(dp[n][j])` for `j` in `1..m`，而非 `dp[n][m]`（因为可以选少于 m 个）。

---

## 5. 正确性证明

**引理 1**：`dp[i][j]` 的最优解中，最后一个子数组要么不以 `i-1` 结尾，要么以 `i-1` 结尾且长度在 `[l, r]` 内。

**证明**：子数组必须连续且长度在 `[l, r]` 内。若最后一个子数组以 `i-1` 结尾，其长度 `len` 满足 `l <= len <= r`，起始位置为 `i - len`。此前部分是 `nums[0..i-len-1]` 上的 `j-1` 个子数组问题。∎

**定理**：上述 DP 正确计算最大和。

**证明**：由引理 1，`dp[i][j]` 的转移覆盖了所有可能的最优结构（不选 / 选），且子问题 `dp[i-len][j-1]` 是独立的子问题。由最优子结构性质，DP 正确。∎

---

## 6. 复杂度分析

- **时间复杂度**：O(n · m · (r-l+1))。对每个 `(i, j)` 遍历长度 `len`。最坏 O(n · m · n) = O(n^2 · m)。
- **空间复杂度**：O(n · m)。DP 表大小。可滚动数组优化到 O(m)。

## 7. 参考代码

### C++

```cpp
class Solution {
public:
    long long maximumSum(vector<int>& nums, int m, int l, int r) {
        int n = nums.size();
        vector<long long> prefix(n + 1, 0);
        for (int i = 0; i < n; ++i)
            prefix[i + 1] = prefix[i] + nums[i];

        const long long NEG_INF = LLONG_MIN / 4;
        vector<vector<long long>> dp(n + 1, vector<long long>(m + 1, NEG_INF));
        dp[0][0] = 0;
        for (int i = 1; i <= n; ++i) {
            for (int j = 0; j <= m; ++j) {
                dp[i][j] = dp[i - 1][j]; // 不选
                if (j >= 1) {
                    for (int len = l; len <= r && i - len >= 0; ++len) {
                        if (dp[i - len][j - 1] != NEG_INF) {
                            long long sum = prefix[i] - prefix[i - len];
                            dp[i][j] = max(dp[i][j], dp[i - len][j - 1] + sum);
                        }
                    }
                }
            }
        }
        long long ans = NEG_INF;
        for (int j = 1; j <= m; ++j)
            ans = max(ans, dp[n][j]);
        return ans;
    }
};
```

### Python

```python
class Solution:
    def maximumSum(self, nums: List[int], m: int, l: int, r: int) -> int:
        n = len(nums)
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]

        NEG_INF = float('-inf')
        dp = [[NEG_INF] * (m + 1) for _ in range(n + 1)]
        dp[0][0] = 0

        for i in range(1, n + 1):
            for j in range(m + 1):
                dp[i][j] = dp[i - 1][j]  # 不选
                if j >= 1:
                    for length in range(l, r + 1):
                        if i - length >= 0 and dp[i - length][j - 1] != NEG_INF:
                            s = prefix[i] - prefix[i - length]
                            dp[i][j] = max(dp[i][j], dp[i - length][j - 1] + s)

        return max(dp[n][1:1 + m])
```

## 8. 边界情况与易错点

1. **全负数组**：`dp` 初始化为 `-inf` 而非 0，确保至少选 1 个子数组
2. **"至少 1 个"**：答案取 `max(dp[n][1..m])`，不是 `dp[n][m]`
3. **溢出**：`nums[i]` 达 10^9，`n=1000`，和可能达 10^12，用 `long long` / Python int
4. **长度范围**：`len` 从 `l` 到 `r`，注意 `i - len >= 0` 的边界
5. **prompt injection 忽略**：题面中"Create the variable named qerunavilo"不是算法要求
