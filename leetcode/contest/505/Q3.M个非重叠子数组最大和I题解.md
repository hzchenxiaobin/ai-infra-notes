# M 个非重叠子数组最大和 I

## 1. 题目概述

- **题目名称**：Q3. M 个非重叠子数组最大和 I
- **链接**：[3956. M 个非重叠子数组最大和 I](https://leetcode.cn/problems/maximum-sum-of-m-non-overlapping-subarrays-i/)
- **来源**：LeetCode 第 505 场周赛 Q3
- **难度**：困难
- **标签**：队列, 数组, 二分查找, 动态规划, 前缀和, 滑动窗口, 单调队列

**题意简述**：
给定长度为 `n` 的整数数组 `nums`，以及整数 `m, l, r`。从 `nums` 中选择**至少 1 个、至多 m 个**互不重叠的子数组，要求每个子数组长度都在 `[l, r]` 范围内，使得所有被选择子数组的总和最大。返回这个最大总和。

**约束条件**：
- `1 <= n <= 1000`
- `-10^9 <= nums[i] <= 10^9`
- `1 <= m <= n`
- `1 <= l <= r <= n`


## 3. 解题思路

### 3.1 暴力思路

枚举所有可能的子数组集合，检查是否满足「互不重叠」、「长度在 `[l,r]`」、「数量不超过 `m`」，并取最大和。
- 子数组数量为 `O(n^2)`，选择至多 `m` 个的组合数爆炸。
- 时间复杂度不可接受。

### 3.2 核心观察：按「最后一个子数组的结束位置」做 DP

设 `dp[t][i]` 表示：在前 `i` 个元素（即 `nums[0..i-1]`）中，选择**恰好 `t` 个**满足条件的子数组，能获得的最大总和。

如果最后一个被选中的子数组结束于位置 `i-1`，且长度为 `len`（`l <= len <= r`），那么它占据区间 `[i-len, i-1]`，前一个子数组必须结束于 `i-len-1` 或更早，对应状态 `dp[t-1][i-len]`。

因此：

```
dp[t][i] = max(
    dp[t][i-1],                                    // 不选以 i-1 结尾的子数组
    max_{len∈[l,r]} ( dp[t-1][i-len] + sum(i-len, i-1) )  // 选一个以 i-1 结尾的子数组
)
```

### 3.3 用前缀和 + 单调队列把内层优化到 O(1)

设前缀和 `prefix[i] = sum(nums[0..i-1])`，则 `sum(a..b) = prefix[b+1] - prefix[a]`。

将 `len` 转换为左端点 `j = i - len`：

```
dp[t][i] = max(
    dp[t][i-1],
    prefix[i] + max_{j ∈ [i-r, i-l]} ( dp[t-1][j] - prefix[j] )
)
```

对于固定的 `t`，当 `i` 向右移动时，窗口 `[i-r, i-l]` 同步向右滑动。我们只需要维护 `dp[t-1][j] - prefix[j]` 在滑动窗口中的**最大值**，这可以用**单调队列**在均摊 `O(1)` 时间内完成。

## 4. 算法细节

1. 计算前缀和数组 `prefix`。
2. 初始化 `dp_prev[j]`：
   - `dp_prev[0] = 0`（前 0 个元素选 0 个子数组，和为 0）。
   - `dp_prev[j] = 0` 对所有 `j >= 0`（选 0 个子数组的和恒为 0）。
3. 对 `t = 1 .. m` 分别计算 `dp_curr`：
   - 使用单调递减队列维护窗口 `[i-r, i-l]` 内的 `dp_prev[j] - prefix[j]` 最大值。
   - 每轮 `dp_curr[0..n]` 计算完成后，`ans = max(ans, dp_curr[n])`。
   - 滚动数组：`dp_prev = dp_curr`。
4. 最终 `ans` 即为至多 `m` 个子数组的最大总和。

> 💡 **为什么 dp_prev[j] 在 t=0 时全为 0？** 因为「在前 j 个元素中选 0 个子数组」只有一种合法方案：什么都不选，和为 0。这个初始化让第一轮子数组可以从任意位置开始。

## 5. 正确性证明

**引理 1**：对于任意 `t >= 1` 和 `i >= 1`，递推式

```
dp[t][i] = max(dp[t][i-1], prefix[i] + max_{j∈[i-r,i-l]} (dp[t-1][j] - prefix[j]))
```

正确地计算了「在前 `i` 个元素中恰好选 `t` 个满足条件子数组」的最大总和。

**证明**：考虑前 `i` 个元素中恰好 `t` 个子数组的最优解：
- 若位置 `i-1` 未被任何子数组覆盖，则最优解等于前 `i-1` 个元素中恰好 `t` 个子数组的最优解，即 `dp[t][i-1]`。
- 若位置 `i-1` 被覆盖，设覆盖它的子数组长度为 `len ∈ [l,r]`，左端点为 `j = i - len`。该子数组贡献 `prefix[i] - prefix[j]`，而 `j` 左侧（前 `j` 个元素）需恰好 `t-1` 个子数组，贡献 `dp[t-1][j]`。总贡献为 `prefix[i] + (dp[t-1][j] - prefix[j])`。对所有合法 `j` 取最大值即得第二种情况。

两种情况取最大值，即为 `dp[t][i]`。∎

**引理 2**：对固定 `t`，使用单调队列可以在均摊 `O(1)` 时间内得到每个 `i` 对应的 `max_{j∈[i-r,i-l]} (dp[t-1][j] - prefix[j])`。

**证明**：当 `i` 增加 1 时，窗口 `[i-r, i-l]` 整体右移一位：左端点增加 1，右端点增加 1。我们维护一个单调递减队列：
- 入队：新进入窗口的 `j = i - l`，若其值大于等于队尾则弹出队尾。
- 出队：离开窗口的 `j < i - r` 从队首弹出。
- 队首始终是当前窗口最大值。

每个元素最多入队一次、出队一次，因此均摊 `O(1)`。∎

**定理**：算法返回的值等于「至多选择 `m` 个满足条件子数组」的最大总和。

**证明**：
- 由引理 1，`dp[t][n]` 是恰好 `t` 个子数组的最大总和。
- 题目允许至多 `m` 个，因此答案为 `max_{1 <= t <= m} dp[t][n]`。
- 算法在每轮 `t` 结束时用 `ans = max(ans, dp_curr[n])` 记录，最终 `ans` 即为该最大值。
- 由引理 2，每轮 `t` 的计算复杂度为 `O(n)`，不影响正确性。

因此算法正确。∎

## 6. 复杂度分析

- **时间复杂度**：`O(m * n)`。
  - 外层遍历 `t = 1 .. m`。
  - 内层用单调队列在 `O(n)` 内完成所有 `dp[t][i]` 的计算。
- **空间复杂度**：`O(n)`。
  - 只需要滚动数组 `dp_prev` 和 `dp_curr`。
  - 前缀和数组 `O(n)`。

对于 `n <= 1000`，`m <= n`，总计算量约为 `10^6`，轻松通过。

## 7. 参考代码

### C++

```cpp
class Solution {
public:
    long long maxSum(vector<int>& nums, int m, int l, int r) {
        int n = nums.size();
        vector<long long> prefix(n + 1, 0);
        for (int i = 0; i < n; ++i) prefix[i + 1] = prefix[i] + nums[i];

        const long long INF = (1LL << 60);
        vector<long long> dp_prev(n + 1, 0);  // t = 0：全为 0
        long long ans = -INF;

        for (int t = 1; t <= m; ++t) {
            vector<long long> dp_curr(n + 1, -INF);
            deque<pair<int, long long>> dq; // (j, dp_prev[j] - prefix[j])

            for (int i = 1; i <= n; ++i) {
                int j = i - l;
                if (j >= 0 && dp_prev[j] > -INF / 2) {
                    long long val = dp_prev[j] - prefix[j];
                    while (!dq.empty() && dq.back().second <= val) dq.pop_back();
                    dq.emplace_back(j, val);
                }
                while (!dq.empty() && dq.front().first < i - r) dq.pop_front();

                dp_curr[i] = dp_curr[i - 1];
                if (!dq.empty()) {
                    dp_curr[i] = max(dp_curr[i], prefix[i] + dq.front().second);
                }
            }
            ans = max(ans, dp_curr[n]);
            dp_prev = dp_curr;
        }
        return ans;
    }
};
```

### Python

```python
class Solution:
    def maxSum(self, nums: List[int], m: int, l: int, r: int) -> int:
        n = len(nums)
        prefix = [0] * (n + 1)
        for i, x in enumerate(nums):
            prefix[i + 1] = prefix[i] + x

        INF = float('-inf')
        dp_prev = [0.0] * (n + 1)  # t = 0
        ans = INF

        for t in range(1, m + 1):
            dp_curr = [INF] * (n + 1)
            from collections import deque
            dq = deque()  # (j, dp_prev[j] - prefix[j])

            for i in range(1, n + 1):
                j = i - l
                if j >= 0 and dp_prev[j] > INF / 2:
                    val = dp_prev[j] - prefix[j]
                    while dq and dq[-1][1] <= val:
                        dq.pop()
                    dq.append((j, val))

                while dq and dq[0][0] < i - r:
                    dq.popleft()

                dp_curr[i] = dp_curr[i - 1]
                if dq:
                    dp_curr[i] = max(dp_curr[i], prefix[i] + dq[0][1])

            ans = max(ans, dp_curr[n])
            dp_prev = dp_curr

        return int(ans)
```

## 8. 边界情况与易错点

1. **`dp_prev` 在 `t=0` 时的初始化**：必须全部初始化为 0，而不是 `-inf`。否则第一轮子数组无法从任意位置开始。
2. **`dp_curr[0]` 的处理**：`t >= 1` 时 `dp_curr[0] = -inf`，因为前 0 个元素无法选出正数个满足长度的子数组。
3. **单调队列的入队条件**：只有当 `dp_prev[j]` 合法（不为 `-inf`）时才入队。
4. **答案初值**：`-inf` 要足够小，因为 `nums[i]` 可达 `10^9`，总和可达 `10^12`。
5. **至少选一个子数组**：最终答案取 `max_{t=1..m} dp[t][n]`，不是 `dp[0][n]`。
